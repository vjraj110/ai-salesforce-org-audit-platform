# AI-Powered Salesforce Org Audit Platform
An AI-powered governance and auditing platform for Salesforce orgs that analyzes Apex, metadata, security configurations, and automation health to identify risks, technical debt, and optimization opportunities.

## Features

- Metadata scanning using Tooling API
- Apex and Trigger analysis
- Extensible rule engine
- Audit findings dashboard
- Org health scoring
- AI-powered recommendations (planned)

- Salesforce Apex
